# simple-flashcards
*A simple CLI flashcard app*

This is a flashcard app that uses command-line interface.

## Features

- easy Flashcard creation
- Flascard editing
- Flashcard shuffling

## Made with

- C++
- STL
- CMake

## Build

How to build: 

```
cmake -B build
cmake --build build
```

If you're using MinGW:

```
cmake -G "MinGW Makefiles" -B build
cmake --build build
```
